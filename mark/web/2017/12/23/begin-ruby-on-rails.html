安装

  1. 安装ruby

  2. 安装sqlite3

     sudo apt-get install sqlite3
 sudo apt-get install libsqlite3-dev

  3. 安装rails

     gem install rails
 rails --version

  4. 使用rails创建博客

     rails new blog

  5. 启动web服务器

     cd blog
 bin/rails server
